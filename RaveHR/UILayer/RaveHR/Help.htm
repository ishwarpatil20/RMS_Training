Projects Module
The Projects module defines an online interface to view, filter and modify project details. Members belonging to RPM, COO, Presales and PM can view the project. Presales can view all active projects in presales. COO and RPM can view all active projects. PM can view all active projects allocated to him/her. Members belonging to RPM, Presales and PM can update the project.
 
 
MRF Module
The MRF module defines an online user interface for Manpower Requisition, Projects and Resource Allocation process; create projects into the system and to raise requisition against a project. Requisition of manpower is usually done by PM or Pre Sales team. RPM then looks for suitable candidates from its internal manpower. If sufficient candidates are not available, newspaper advertisements, referrals, and other forms of recruitment are resorted.
